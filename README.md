# AI388 CNS projects:

## [P001]: Secure student face recognition system. 
This system adopts deep learning technology to develop an intelligence face recognition system for students. This, for example, can be used to track student attendance, or perform authentication. The system should provide a level of security where a malicious person cannot use a picture of the student or a crafted mask to bypass the system.

## [S005] Phishing Email Detector using Machine Learning.
This project develops a system that detects phishing emails using machine learning and text-based features. You will have to extract indicators such as suspicious URLs, keywords, and email structure, then train and evaluate classification models. A small interface will be designed to demonstrate phishing probability prediction.
